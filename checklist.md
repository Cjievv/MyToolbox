# MyToolBox 开发工具集 - 验收清单

## 功能验收

### 1. JSON 格式化工具

- [ ] 格式化功能正常 - 输入 JSON 字符串，点击格式化按钮后输出带缩进的格式
- [ ] 压缩功能正常 - 点击压缩按钮后输出单行压缩格式
- [ ] 验证功能正常 - 输入无效 JSON 时显示错误状态
- [ ] YAML 转换正常 - 点击"转 YAML"按钮可转换为 YAML 格式
- [ ] 树形预览正常 - 点击"树形"标签页可查看可折叠的层级结构
- [ ] 缩进选择正常 - 可选择 2 空格或 4 空格缩进
- [ ] 复制功能正常 - 点击复制按钮可复制结果到剪贴板
- [ ] 下载功能正常 - 点击下载按钮可导出为 .json 文件
- [ ] 粘贴功能正常 - 点击粘贴按钮可从剪贴板读取内容

### 2. 文本差异工具

- [ ] 双栏输入正常 - 左右两侧可分别输入文本
- [ ] 差异计算正常 - 可计算并显示差异
- [ ] 统计显示正常 - 正确显示新增、删除、修改的行数
- [ ] 并排视图正常 - 可切换到并排视图模式
- [ ] 行内视图正常 - 可切换到行内视图模式
- [ ] 忽略空白正常 - 勾选后可忽略空白符差异
- [ ] 文件导入正常 - 可从文件读取内容到输入框
- [ ] 复制功能正常 - 可复制两侧的文本内容
- [ ] 导出功能正常 - 可导出差异结果为文本文件

### 3. SQL 格式化工具

- [ ] 格式化功能正常 - 输入 SQL 语句后可格式化为易读格式
- [ ] MySQL 方言支持 - 可选择 MySQL 方言进行格式化
- [ ] PostgreSQL 方言支持 - 可选择 PostgreSQL 方言进行格式化
- [ ] SQLite 方言支持 - 可选择 SQLite 方言进行格式化
- [ ] MariaDB 方言支持 - 可选择 MariaDB 方言进行格式化
- [ ] PL/SQL 方言支持 - 可选择 PL/SQL 方言进行格式化
- [ ] 关键字大写 - UPPER 模式将关键字转为大写
- [ ] 关键字小写 - lower 模式将关键字转为小写
- [ ] 关键字保持 - preserve 模式保持原样
- [ ] 2 空格缩进 - 可选择 2 空格缩进
- [ ] 4 空格缩进 - 可选择 4 空格缩进
- [ ] 压缩功能正常 - 可将 SQL 压缩为单行
- [ ] 文件导入正常 - 可导入 .sql / .txt 文件
- [ ] 复制功能正常 - 可复制格式化结果
- [ ] 下载功能正常 - 可导出为 .sql 文件

### 4. Markdown 编辑器

- [ ] 编辑功能正常 - 左侧可编辑 Markdown 源码
- [ ] 实时预览正常 - 右侧显示实时渲染的 HTML
- [ ] 粗体工具正常 - 点击工具栏粗体按钮可插入 **text**
- [ ] 斜体工具正常 - 点击工具栏斜体按钮可插入 *text*
- [ ] 标题工具正常 - 点击工具栏标题按钮可插入 ##
- [ ] 列表工具正常 - 点击工具栏列表按钮可插入 -
- [ ] 引用工具正常 - 点击工具栏引用按钮可插入 >
- [ ] 代码块工具正常 - 点击工具栏代码块按钮可插入 ```
- [ ] 链接工具正常 - 点击工具栏链接按钮可插入 [text](url)
- [ ] 图片工具正常 - 点击工具栏图片按钮可插入 ![alt](url)
- [ ] 表格工具正常 - 点击工具栏表格按钮可插入表格模板
- [ ] 大纲导航正常 - 左侧显示标题大纲，点击可跳转
- [ ] GFM 支持正常 - 支持任务列表、删除线等 GFM 语法
- [ ] 文件导入正常 - 可导入 .md / .txt / .markdown 文件
- [ ] 文件导出正常 - 可导出为 .md 文件
- [ ] 复制功能正常 - 可复制 Markdown 源码

### 5. 文件转换工具

- [ ] JSON→YAML 转换正常
- [ ] YAML→JSON 转换正常
- [ ] JSON→CSV 转换正常 (数组格式)
- [ ] CSV→JSON 转换正常
- [ ] Markdown→HTML 转换正常
- [ ] 卡片选择正常 - 点击卡片可切换转换类型
- [ ] 错误提示正常 - 转换失败时显示错误信息
- [ ] 复制功能正常 - 可复制转换结果
- [ ] 导出功能正常 - 可导出转换后的文件
